# 瞄点跟踪 + 循迹小车 — STM32

基于嘉立创天空星 STM32 的通用移动机器人平台，支持循迹、云台追踪、激光控制等多种应用。

## 硬件

| 部件 | 说明 |
|------|------|
| 主控 | STM32 天空星开发板 |
| 视觉 | K230 摄像头模块（UART 通信） |
| 云台 | 步进电机 + 舵机 |
| 执行器 | 激光笔、直流电机 |
| 驱动 | 支持阿克曼模型 / 四轮驱动模型 |

## 功能

- **循迹模式**：灰度传感器 + PID 速度/方向控制，支持阿克曼和四轮差速两种模型
- **瞄点追踪模式**：K230 识别光点坐标 → UART 传给 STM32 → 步进电机云台跟踪
- **激光控制**：云台追踪时可控制激光笔指向目标

## 使用

用 Keil MDK 打开项目编译下载，通过串口切换工作模式。
