# 瞄点跟踪 + 循迹小车 — STM32

基于嘉立创天空星 STM32 的通用移动机器人平台，支持循迹行驶和云台瞄点追踪两种工作模式，兼容阿克曼转向和四轮差速两种底盘模型。

## 硬件方案

| 部件 | 型号 | 说明 |
|------|------|------|
| 主控 | STM32 (天空星开发板) | ARM Cortex-M 系列 |
| 视觉 | K230 AI 摄像头模块 | UART 通信，识别光点坐标 |
| 云台 | 步进电机 + 舵机 | 二维云台，追踪光点 |
| 执行器 | 激光笔 | 指示追踪目标 |
| 电机驱动 | TB6612 / 其他 | 支持阿克曼和四轮驱动 |
| 传感器 | 灰度传感器 | 循迹模式检测黑线 |

## 功能特性

### 循迹模式

- 灰度传感器检测赛道黑线
- PID 速度闭环 + PID 方向闭环
- 支持**阿克曼转向模型**（前轮转向 + 后轮驱动）
- 支持**四轮差速模型**（左右轮差速转向）
- 少量代码修改即可切换底盘模型

### 瞄点追踪模式

1. K230 摄像头采集图像，识别目标光点
2. 通过 UART 将光点坐标 (`@errorX#errorY%`) 发送给 STM32
3. STM32 控制步进电机云台追踪光点
4. 舵机辅助微调激光笔指向

### 外设支持

项目包含多种外设驱动，可根据需要组合使用。

## 软件架构

- STM32CubeMX 初始化配置
- HAL 库外设驱动
- PID 控制算法（速度环 + 方向环）
- K230 UART 通信解析
- 云台步进电机控制

## 使用

用 Keil MDK 打开工程编译下载，通过串口或按键切换工作模式。
